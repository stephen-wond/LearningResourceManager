Here we are using a HTTP PUT to add a new resource to the database. The inputted data could pose a security risk, so it will need to be validated and also it should not be allowed to reach the database in its raw form. After the resource has been added the cache will need to be updated to reflect the change, this will need to be done each time the database is edited to ensure that the latest data is being displayed and is also avaliable for edit/delete
